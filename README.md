# Qahwa

Good ideas start with brainstorming. 
Great ideas start with coffee. 

Qahwa is a social netwark blog app made for AND course at VIA University Collage. 
A virtual place, where coffee addicteds can share there thoughts, photos and good coffee recipes maybe?


## MoSCoW requirements

### Must have:
- [ ] An authentication system
- [ ] Create postes (photos and optional descriptons)
- [ ] Follow others users functionality
- [ ] Feed view, where you can see your own postes, and the users you follow 
- [ ] Like and comment functionality
- [ ] Create, edit or delete profile picture 
- [ ] Search functionality in order to find a user by his name. 
	
### Should have:
- [ ] Friendly user interface
- [ ] Counter of likes and comments

### Could have:
- [ ] Private or group chat
- [ ] Videos uploading
- [ ] Buttons animations
